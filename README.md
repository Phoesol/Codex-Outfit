## 🎯 解决什么问题？

在酒馆角色扮演中，AI 经常自己脑补服装描写，导致人设崩坏。比如明明设定是西装总裁，AI 下一句就换上了睡衣。

穿搭管理器把穿搭信息注入到对话上下文，AI 每次描写服装时都有据可依，再也不会乱编。

---

## ✨ 核心功能

### 👗 真实衣柜管理
上传你的穿搭照片，配上文字描述、风格标签（学院/街头/甜美…）、季节标签（春夏秋冬）、场景标签（外出/睡前/约会/家居…）。AI 会根据你穿的这一套来描写。

### 🌐 AI + 世界书联动
内嵌 20 种现代风格（纯欲、甜酷、千禧Y2K、韩系女团、旗袍、新中式…）+ 5 款内衣的详细穿搭描述。一键随机，天天不重样。配合 💎uu现代v2.1 / 🦋uu内衣v1.0 世界书使用效果更佳。

### 📋 批量导入
把照片发给 ChatGPT/Claude → 得到描述 → 复制粘贴进面板 → 自动解析名称/分类/风格/季节/场景/描述 → 一键创建 N 套穿搭。面板内置「复制提示词」按钮，不用手打。

### 🎲 场景快捷切换
底部一排按钮：`[睡前] [外出] [约会] [家居] [办公] [运动]`，一键换装。勾选「仅世界书」就只看世界书风格，不碰真实衣柜。

### 🎰 启动自动随机
每次打开酒馆自动从世界书里穿一套，右下角弹 toast 告诉你今天穿了什么。左下角状态栏实时显示「穿着：xxx」。

### 👤 User / 角色独立
User 和不同角色各有独立衣柜，各自激活各自的穿搭，互不干扰。

### 🏷️ 分类管理
支持自定义分类（睡衣/制服/常服/外出服…），AI 生成或粘贴的分类自动加入分类栏。

## 📦 安装

1. 下载 `Outfit-Manager` 文件夹
2. 放入 `SillyTavern/public/scripts/extensions/third-party/`
3. 重启酒馆 → 扩展管理 → 启用 → 完成

---

## 🚀 快速上手

### 场景一：我有自己的穿搭照片

```
1. 点击「批量添加」
2. 上传照片 + 点「复制提示词」→ 粘贴给外部 AI
3. AI 返回描述 → 粘贴回面板
4. 点击「创建 N 套」→ 搞定
```

### 场景二：我只想 AI 随机生成

```
1. 勾选底部「仅世界书」
2. 点击场景按钮（睡前/外出…）→ 弹出完整穿搭描述
3. 满意点「确认」→ 穿上
4. 下次打开酒馆自动换一套新的
```

### 场景三：有时穿自己的，有时随机

```
1. 平时勾「仅世界书」→ 天天随机
2. 想穿自己某套时 → 去衣柜点激活
3. AI 按激活的那套描写
```

## 💡 Tips

- 场景标签建议用简短中文：`睡前` `外出` `约会` `家居` `办公` `运动`
- 世界书风格描述支持直接复制去文生图
- 多选模式可以批量管理（删除/移动分类）
